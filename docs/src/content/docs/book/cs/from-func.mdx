Type system

In TACT we meet `Bool{:tact}` (classical boolean with true and false values), `Address{:tact}` (standard smart contract address in TON Blockchain), `String{:tact}` (immutable text strings) & `StringBuilder{:tact}` (helper type that allows you to concatenate strings in a gas-efficient way). You can specify the size or persistent state variables with `as{:tact}` keyword:
